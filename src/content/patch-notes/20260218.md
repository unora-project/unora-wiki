Launcher Update – Synchronized Radio System

I’ve added a synchronized radio system directly into the launcher.

Queue up YouTube videos and listen together with everyone using the new Radio feature.

👉 Restart your launcher and it will update automatically.

How it works

Paste any YouTube link into the Radio window and add it to the queue

The server grabs the video + metadata and distributes it to all clients

When a track loads, everyone's player waits until all listeners are ready before it starts

When a track ends, the next one plays automatically once all clients report finished

If you join mid-song, you’ll drop in at the correct timestamp

Features

🎵 Shared queue visible to all listeners

⏭️ Vote to skip — majority vote advances the track

👥 Live listener count

Open the launcher and click the music note icon at the top to jump in and try it out.

For everyone

You’ll see a green 🎵 AUTO-FILLING · [Playlist name] banner in the queue panel when that’s happening.
