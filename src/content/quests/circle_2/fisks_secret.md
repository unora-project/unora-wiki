# Fisk's Secret

## Rewards

- Varied experience (605,810 experience if you follow this guide)
- Legend mark:
    - If you keep Fisk's secret: "Inspired Fisk to Reach for the Stars"
    - If you betray Fisk: "Betrayed Fisk's Trust"

!!! note

    This quest has multiple routes with different outcomes. This guide follows the "pure" route that honors Fisk's request — also the most expensive. Have about **60,000 gold** on you.

## Walkthrough

### Fisk

1. Head to Suomi and speak to Fisk, the Suomi Special Skills Master **(12,18)**
2. Select "Fisk's Secret" to start the quest
3. Select "Will you tell me who it is?"
4. Select "You have my word."
5. Select "That's so sweet."
6. Select "I will try."
7. Select "I will ask."

> Fisk wants you to speak to Thulin at the Suomi theater and gather 5 Water Lilies.

### Thulin, Garamonde Theatre

1. Head to the Suomi Theater **(13,30)** and speak to Thulin **(2,9)**
2. Select "Water Lilies"
3. Select "I only need five."
4. Select "Sure, what am I getting?"
5. Select "I can't say, but I will be back."

> Thulin wants 10 Wine from Viveka in exchange for the Water Lilies.

6. Head to Viveka in Suomi **(82,15)** and buy **10 Wine**
7. Return to Thulin to trade the Wine for 5 Water Lilies

### Back to Fisk

1. Return to Fisk with the Water Lilies
2. Select "No, I haven't. Your secret is safe."
3. Select "Where can I get them?"
4. Select "I will go ask."

> Fisk asks for 5 Petunias from Eeva.

### Eeva

1. Head to Eeva in Suomi **(32,55)**
2. Select "Petunias"
3. Select "They really are."
4. Select "Yes, I would like some of these Petunias."
5. Select "I can't say, but I would like five please."
6. Select "I will get them right away."

> Eeva wants **20 Grapes** from the Suomi farm in exchange.

7. Head to Alvar, the Suomi Grape Farmer **(62,41)**
8. Choose to either farm grapes yourself or buy them at 500 gold each — you need 20
9. Return to Eeva with the 20 Grapes
10. Select "Petunias"
11. Select "Yes I have them."
12. Return to Fisk with the Petunias
13. Select "Fisk's Secret"
14. Select "Eeva gave me them."
15. Select "Secret is still safe."
16. Select "I can try."
17. Select "I won't tell her anything."

### Vivianne

> Fisk wants you to get a Pink Rose from Vivianne.

1. Head to Vivianne at the Suomi entrance in Suomi Village Way **(21,4)**
2. Select "Pink Rose"
3. Select "I was, I'm sorry."
4. Select "I need one for a friend."
5. Select "You're right, but I can't say more."
6. Select "I've said too much. What can I do for one?"
7. Select "He is making a bouquet."
8. Select "Deal" — you donate **20,000 Gold** for a Pink Rose
9. Return to Fisk
10. Select "Fisk's Secret"
11. Select "Yes, here it is."
12. Select "She knows a little."

### Viveka's Bouquet

> Fisk asks you to have Viveka make a bouquet from the flowers.

1. Head to Viveka **(82,15)**
2. Select "Bouquet"
3. Select "Yes, I have the flowers."
4. Select "I can't say."
5. Select "I'm sure, sorry."
6. Select "That is fine, I will pay." — give **25,000 Gold**
7. Select "\*Hand over the flowers and 25,000 gold\*"

!!! warning

    You must wait **1 real life hour** for Viveka to finish arranging the bouquet.

8. After 1 hour, speak with Viveka and select "Bouquet"
9. Select "I apologize but I still can't say."

> Viveka hands you a **Special Bouquet**.

### Delivering to Bertil

1. Return to Fisk
2. Select "Fisk's Secret"
3. Select "Yes, here it is."
4. Select "I will deliver it to her."
5. Head to Bertil, the Suomi White Magic Master **(16,86)**
6. Select "Special Delivery"
7. Select "\*Hand over the Bouquet\*"
8. Select "I can't say. You have a secret admirer."
9. Select "I'll tell him to come see you."
10. Select "Your admirer wants you to find him."

> Bertil says she has no time to search for her admirer.

### Relaxing Fisk

1. Return to Fisk
2. Select "Fisk's Secret"
3. Select "Bertil wants to see who sent the flowers."
4. Select "She does not know it is you."
5. Select "The flowers are from a secret admirer."
6. Select "She asked if I could relay a message."
7. Select "Please don't make her search. Come to her."
8. Select "She very well may, but you have to try."
9. Select "Sure, what is it?"

> Fisk wants **10 Brandy** to calm his nerves before confessing.

10. Head to Viveka and purchase **10 Brandy** for 400 gold
11. Return to Fisk and select "Fisk's Secret"

!!! warning

    You must wait another **1 real life hour** for Fisk to visit Bertil.

### The Outcome

1. Speak with Fisk
2. Select "Fisk's Secret"
3. Select "I'm so happy for you." to complete the quest
