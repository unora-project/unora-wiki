# Supply Loures Army

## Requirements

- [Sick Child](/quests/circle_2/sick_child) quest must be completed

## Rewards

- Chance of learning Sweet Buns cooking recipe
- Legend mark:
    - If you cured the princess: "Supplied the Loures Army"
    - If you killed the princess: "Thrown out of Loures Castle by the Knights" (dark orange)

??? info "\"Good run\""

    This walkthrough assumes you cured the princess in the Sick Child quest. Instructions may differ if you chose to kill her.

## Walkthrough

### Knight Thibault

1. Head to the Loures Throne Room (Loures 3 Floor Office)
2. Speak to Knight Thibault and select "The Sick Child" to start the quest
3. Select "What can I do to help?"
4. Select "I will go speak to him."

> The princess is feeling better, but the King is spending more time with her. With his absence, the Loures army is struggling with supplies. Knight Alistair is leading the army and needs help.

### Knight Alistair

1. Exit Loures Castle and run to the left side of the castle **(1,33)**
2. In the Loures Training Camp, speak to Knight Alistair
3. Select "The Sick Child"
4. Select "Knight Thibault sent me to help."
5. Select "What materials do you need?"
6. Select "Okay Polished Iron, how much?"
7. Select "What else?"
8. Select "You need more?"
9. Select "No problem, I'll be back."

> Knight Alistair needs:
>
> - **10 Polished Iron Bars**
> - **5 Pristine Ruby**
> - **10 Exquisite Cotton**

### Completing the Quest

1. Collect the requested items
2. Return to Knight Alistair and select "The Sick Child"
3. Select "Yes, I did."

> Knight Alistair tells you to report back to Knight Thibault.

4. Head back to the Loures Throne Room
5. Speak to Knight Thibault and select "The Sick Child"
6. Select "It was my pleasure."
7. Select "Thank you Knight Thibault" to complete the quest
