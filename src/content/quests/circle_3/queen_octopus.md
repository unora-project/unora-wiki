# Queen Octopus

## Rewards

- 700,000 Experience
- Legend mark: Killed the Queen Octopus of Karlopos Island

## Walkthrough

### Octopus Liver

1. Head to Rucesion Inn **(51,42)**
2. Speak to Maria
3. Select "Queen Octopus?"
4. Select "Do you know anything about her?"
5. Select "Interesting. Could I have it by chance?"
6. Select "Sure, where can I find them?"
7. Select "I'll be back with the livers."

> Maria asks you to get her **5 livers** from Octopus on Karlopos Island.

8. Collect 5 Octopus livers from Karlopos Island
9. Return to Maria and select "Queen Octopus"
10. Select "I have the livers you asked for."
11. Click "Next"

> Maria hands you a **Red Pearl** and a clue about a secret entrance to Karlopos Island North.

### Finding Maria's Brother

!!! warning "Party required!"

1. Head to Karlopos Island
2. Go north 2 maps to arrive on Karlopos Island East
3. In the rocky area at the entrance, go to **(7,38)** to find the Karlopos Secret Passage
4. Speak to Bret
5. Select "You must be Maria's brother."
6. Select "Not exactly, but she is worried about you."
7. Select "Same as you. I'm Looking for the Queen Octopus."
8. Select "Just traps? I can handle that."
9. Select "Thank you. Will you be joining us?"
10. Select "Sounds good. Thank you."

> Bret warns you the path ahead is full of deadly traps. To find the queen, you'll need a **pendant** from the end of this path, along with the Red Pearl from Maria.

### Trap Room and Trials

!!! warning

    The trap room ahead is very deadly. Bring a Rogue if possible — otherwise, you'll need constant healing. Traps deal heavy damage, but once triggered, the tile is safe.

1. Walk north one map and select "Yes, we are ready." to be teleported to Karlopos Island Long Beach
2. Walk forward very slowly, in a line. If you have a Rogue, ask them to cast **Detect Traps**
3. Make it to the end and rest up

!!! danger "Combat incoming!"

4. Enter the next map and select "Yes, we are ready." — slay all **5 Spores** (Trial 1)
5. Proceed and select "Yes, we are ready." — slay all **5 Gogs** (Trial 2)
6. Proceed and select "Yes, we are ready." — slay all **5 Krakens** (Trial 3)
7. Proceed and select "Yes, we are ready." — slay all **6 Octopus** (Trial 4)
8. Proceed and select "Yes, we are ready."
9. Walk to **(8,5)** to pick up the **Coral Pendant**
10. Return to Maria at the Rucesion Inn
11. Select "Queen Octopus?"
12. Select "No problem."
13. Select "Sweet! I'll be on my way."

> Maria says Bret left a message: to summon the queen, bring the Coral Pendant and Red Pearl to the coastline in Karlopos Island North.

### The Queen's Lair

!!! danger "Boss room ahead!"

1. Head back to Karlopos Island
2. From Karlopos Island South, go up 2 maps and left 2 maps
3. Head to **(21,5)** and rest up
4. Walk forward into the coastline and select "We are ready."
5. Defeat the **Queen Octopus** (she summons other Octopus to fight alongside her)
6. Return to Maria at the Rucesion Inn
7. Select "Queen Octopus?"
8. Select "I did."
9. Select "It was nothing." to complete the quest
