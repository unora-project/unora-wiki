# Purple Whopper Scales

## Rewards

- 25,000 Experience
- 8% chance to obtain "Loved by Undine Mundanes" legend mark

## Cooldown

22 hours

## Walkthrough

1. Head to Undine and visit Undine Storage **(86,17)**
2. Speak with Ayumi and select "Purple Whopper Scales"
3. Select "Always am! What medication?"
4. Select "I'll catch a Purple Whopper for you."
5. With the fishing profession, fish up a **Purple Whopper**
    - This is a rare fish! It could take many hours before you catch one
6. Return to Ayumi with the Purple Whopper to complete the quest
