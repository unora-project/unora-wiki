# Red Potion

## Rewards

- 15,000 Experience

## Cooldown

22 hours

## Walkthrough

1. Speak to Bertil in Suomi **(16,86)**
2. Select "Red Potion" to start the quest
3. Select "What do you mean?"
4. Select "I will go get one then."

> Bertil is requesting you return with a Red Potion.

!!! tip "How to get a Red Potion"

    Red Potion (or "Health Potion") is made from an **Apple**, a **Passion Flower**, and an **Empty Bottle**.

    - Apples can be found from monsters in Mileth Crypt 1
    - Passion Flowers are found randomly through Unora or exchanged in the Piet Alchemy Lab **(18,35)**
    - Empty Bottles can be purchased from Mathis in the Piet Alchemy Lab **(18,35)**

    Find another player who's taken the Alchemist profession and ask them to craft one. If they ask, it's the regular Health Potion, not the small one!

5. Return to Bertil with the Red Potion ("Health Potion") to complete the quest
